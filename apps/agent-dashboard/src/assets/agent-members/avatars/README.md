# Agent and Organization portraits

This initial set contains eight generated editorial portraits shared by Agent
Team MemberRun and Organization surfaces:

- `workspace-architect.png`
- `research-verifier.png`
- `technical-lead.png`
- `implementation-engineer.png`
- `product-strategist.png`
- `security-reviewer.png`
- `finance-operations.png`
- `content-creative.png`

The images are presentational identity aids. Actor name, role, provider, and
runtime status remain authoritative text. `portraitFor()` performs role-first
selection and then a stable identity hash, so unknown actors keep one face
across Mission, Agent Team, Member Focus, and Organization views.

Generation contract: square 512px production derivative, warm ivory backdrop,
semi-realistic editorial illustration, consistent head-and-shoulders crop,
natural skin texture, no text/logo/watermark, and a safe circular-crop margin.
