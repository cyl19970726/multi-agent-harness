# Standing Agent avatar prompt set

Shared prompt for every asset:

```text
Use case: stylized-concept.
Asset type: Company OS Standing Agent avatar.
Create one sophisticated editorial 3D portrait icon of an AI agent, square
composition, centered head and shoulders, crisp circular silhouette, warm ivory
background, soft studio lighting, matte ceramic and subtle brushed-metal
materials, graphite details, premium restrained product-design aesthetic.
Professional and intelligent, not childish, not a mascot, not a generic robot
emoji. No text, no letters, no logo, no watermark, no border, no scenery.
```

Per-Agent additions:

- Company Lead: calm coordinator, deep sage and warm gold accents, subtle
  orchestration ring behind the head, balanced symmetrical design.
- Document Architecture Agent: thoughtful information architect, slate blue
  accents, subtle layered-page and structural grid motif.
- Finance Agent: precise trusted controller, forest green and brass accents,
  subtle ledger grid motif, composed analytical expression.
- Content Strategy Agent: expressive editorial strategist, coral and warm sand
  accents, subtle flowing page/speech motif, curious confident expression.
- Trademark Agent: vigilant legal/IP specialist, burnt orange and graphite
  accents, subtle shield and registration motif, focused expression.
