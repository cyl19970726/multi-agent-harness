# Lead Agent Workspace · V2 expected prompt

Use case: ui-mockup. Asset: shippable desktop web-app product mockup, 1536×1024.

Create a premium Lead Agent collaboration workspace inspired by the strongest
modern AI coding/workspace interfaces, adapted for long-lived company
coordination. This is not an Agent Team execution console. The central surface
must feel alive, conversational, precise, and deeply linked to company objects.

Shared Company OS sidebar with Organization active. Compact header identifies
“Company Lead”, “Lead Agent”, explicit Available status, reports to Brand Owner,
and a subtle overflow menu. Use three columns after the sidebar: a narrow 210 px
direct-report rail, a broad central collaboration stream, and a 300 px Context
Rail.

Direct-report rail: Document Architecture Agent, Finance Agent, Content
Strategy Agent, proposed Trademark Agent; typed badges and small pressure
signals, no model/provider info. Central stream: Human message “Prepare Brand A
trademark filing”; Lead response that created Milestone “Application package
ready” and assigned WorkItems; a rich WorkItem update from Trademark Agent with
linked filing package; Finance Agent blocker “Waiting for Human approval”; a
Mission summary card for similarity research showing 3 Waves completed and
temporary Agent Team outcome, folded rather than raw logs; a governance proposal
to add Trademark Agent. Use source labels and timestamps. Bottom composer says
“Message Company Lead…” with attach Document, create WorkItem, and send actions.
Optionally show one tiny transient “Lead is reviewing…” presence line, visually
separate from durable history.

Context Rail: “Needs attention” with the pending CNY 3,000 decision; “Current
responsibilities” with three Milestones; “Authority” listing assign WorkItems,
start Mission/Team/Workflow, propose Agent, and Human-required boundaries;
“Related modules” Trademark Management and Content Operations.

Visual language: warm ivory, milk-white surfaces, graphite type, coral primary,
sage/amber semantic detail, dense but breathable, refined dividers, no stock
cards. Constraints: no raw chain-of-thought, no persisted thinking, no provider
session, no TeamRun lifecycle as durable identity, no Project, no purple, no
watermark, no browser chrome.
