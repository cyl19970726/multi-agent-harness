# Company OS V2.1 visual review

```text
status: direction materially improved; Human review pending
scope: Home, Docs, Lead Agent Workspace
```

## Result

- Home now has one unmistakable Human-decision focal area, an open Milestone
  sequence, and a recognisable Lead Agent brief instead of a KPI strip.
- Docs retains its full workbench capability while making the document paper,
  authored content, inline live views, maintenance identities, and margin
  collaboration visually primary.
- Lead Agent retains direct reports, WorkItems, Milestones, Mission summary,
  proposal, authority, Needs Attention, and composer while reducing the sense
  of a boxed audit feed.
- all three pages use the canonical navigation groups and integrate distinct
  Agent identity into the visual direction.

## Truth boundary

Generated names, dates, and small avatar renderings remain visual intent, not
canonical data. The implementation must use typed ActorRef records, the stable
avatar files under `identity/avatars/`, authoritative July 2026 fixture data,
and the Company OS object contracts. It must not OCR or hard-code generated
mockup text.

## Next gate

Human review decides whether V2.1 establishes the desired workbench quality.
Only after that decision should the design expand to Organization, Work,
Business Module, typed WorkItem, Milestone, Approval, Finance, and one-time
execution pages or proceed to interactive HTML prototypes.
