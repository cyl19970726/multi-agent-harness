# 0005: Self-Hosting First

## Decision

The first MVP pilot is this repository managing its own development.

## Rationale

The harness must prove it can improve its own docs, schemas, CLI, CI/CD,
provider integration, and Dashboard before it coordinates another project's
strategy work.

## Consequences

Earning Engine remains the first project adapter pilot, but it should follow
the self-hosting proof rather than become the only source of product
requirements.
