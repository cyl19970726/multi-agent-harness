# Agent Team war room prompt

Use case: `ui-mockup`

Design a high-fidelity 1536×1024 Agent Team war room entered from Mission Wave 2. A first-time operator must understand who is working, each member's current assignment, collaboration and handoffs, the blocked member, and the human action required.

Use a connected team presence rail rather than separate member cards. Make a chronological semantic activity stream the dominant surface: Host assignment, Research action, handoff, Backend action summary/artifact, QA review request, and Host decision. Attach “QA approval required” / “Review request” to the QA lane and matching event. Keep the Wave, gate readiness, selected member, and resources in a light contextual rail. Preserve a reachable message composer.

Match the Mission visual system: warm white, charcoal, coral, electric blue, emerald, amber, real portraits, sparse hairlines, selective radii, and small shadows only for floating intervention controls. Show subtle live rail and event-spine motion concepts.

Exclude member-card grids, dashboards/charts, task graph, kanban, raw terminal dumps, raw thinking, giant warning bars, neon, glassmorphism, and modals.

The complete generation prompt and reference provenance are recorded in the creating Codex task; this concise contract is the durable implementation source.
