# Mission Wave canvas prompt

Use case: `ui-mockup`

Design a high-fidelity 1536×1024 Mission execution control plane for Star Harness. A first-time operator must immediately understand what is happening now, what completed, what needs attention, and what happens next. Mission is durable intent containing ordered lightweight Waves; each Wave has an objective, executor, outcome/artifacts, and gate.

Use one continuous editorial execution journey rather than a card grid. Completed Wave 1 is quiet, current Wave 2 expands into a seamless work surface, and planned Wave 3 is subdued. Attach a compact coral “QA approval required” / “Review gate” decision control to the current Wave gate. Use a lightweight contextual rail for Mission brief, current live activity, selected Wave, and gate readiness.

Visual language: warm-white canvas, charcoal typography, coral action accent, electric blue active execution, emerald accepted, amber waiting, sparse hairlines, selective radii and shadows. Include a subtle active progress trace and a tiny Live connection pulse.

Exclude oversized cards, card grids, charts, task graph, Goal/GoalPhase, generic Project, raw thinking, modal dialogs, neon, glassmorphism, and global gradients.

The complete generation prompt and reference provenance are recorded in the creating Codex task; this concise contract is the durable implementation source.
